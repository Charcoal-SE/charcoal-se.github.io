Spam Last Week

Route Name

Spam Last Week

Route URL

/api/spam/last_week

Description & Usage

This route allows you to retrieve data about how many posts were marked as spam and were true positives, for each website in the past week.

Example

Request:

HTTP/1.1 GET /api/spam/last_week
